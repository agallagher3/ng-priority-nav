Priority Nav Demo

A very simple directive, with simple options

Simple example

      <nav class="navbar navbar-light bg-faded">
        <ul class="nav" ng-priority-nav ng-priority-nav-more-link-class="fa fa-list">
          <li ng-repeat="navItem in ctrl.navItems">
            <a ng-bind="navItem.title"> </a>
          </li>
        </ul>
      </nav>

Dynamic example

      <nav class="navbar navbar-light bg-faded">
        <ul class="nav" ng-priority-nav="{{ctrl.navItems2}}" ng-priority-nav-more-link-class="fa fa-list">
          <li ng-repeat="navItem in ctrl.navItems">
            <a ng-bind="navItem.title"> </a>
          </li>
        </ul>
      </nav>
